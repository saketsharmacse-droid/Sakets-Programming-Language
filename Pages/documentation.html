Your First Program

Let's start with the classic "Hello World" program, but with a Saket's Language twist. Because why be boring when you can be awesome?

hello_world.saket

// Your first Saket's Language program
bhai naam = "World"
bol "Hello, " + naam + "! üëã"
bol "Welcome to Saket's Language!"

// Let's do some quick math too
calculate karo 2 + 2
bol "By the way, 2 + 2 = " + result

Expected Output:

Hello, World! üëã
Welcome to Saket's Language!
By the way, 2 + 2 = 4

Breaking It Down

bhai naam = "World"

Creates a variable named 'naam' with the value "World". Yes, we use 'bhai' instead of 'var' or 'let' because friendship matters!

bol "Hello, " + naam + "! üëã"

Prints text to the console. 'bol' means "speak" in Hindi - much more friendly than 'print' or 'console.log'!

calculate karo 2 + 2

Performs mathematical calculations. The result is automatically stored in the 'result' variable.

Variables & Data Types

In Saket's Language, declaring variables is as easy as introducing a friend. We use 'bhai' (brother) because every variable is part of our coding family!

Variable Declaration

Variables Example

// Different types of variables
bhai name = "Saket"           // String
bhai age = 25                 // Number
bhai isAwesome = true         // Boolean
bhai hobbies = ["coding", "gaming", "music"]  // Array
bhai person = {               // Object
    naam: "Saket",
    age: 25,
    city: "Mumbai"
}

// Display all variables
bol "Name: " + name
bol "Age: " + age
bol "Is Awesome: " + isAwesome
bol "Hobbies: " + hobbies[0]
bol "Person: " + person.naam

Supported Data Types

  • String - Text values
  • Number - Integers and decimals
  • Boolean - true/false values
  • Array - Lists of values
  • Object - Key-value pairs

Variable Rules

  • Always start with bhai
  • Variable names can contain letters, numbers, _
  • Cannot start with numbers
  • Case sensitive (naam ‚â† Naam)
  • No spaces in variable names

Conditionals

Making decisions in code should feel natural. That's why we use 'agar' (if) and 'warna' (else) - words you'd actually use in conversation!

Conditional Logic Example

// Simple if-else
bhai age = 20
bhai name = "Rahul"

agar (age >= 18) {
    bol name + " is an adult! üéâ"
    agar (age >= 21) {
        bol "Can drink in the US too!"
    } warna {
        bol "But not old enough for US bars üç∫"
    }
} warna {
    bol name + " is still a kid!"
    calculate karo 18 - age
    bol "Wait " + result + " more years"
}

// Multiple conditions
bhai score = 85

agar (score >= 90) {
    bol "Excellent! A+ grade üåü"
} warna agar (score >= 80) {
    bol "Great job! A grade üëè"
} warna agar (score >= 70) {
    bol "Good work! B grade üëç"
} warna {
    bol "Keep trying! You'll get there üí™"
}

Best Practices

Follow these guidelines to write clean, efficient, and maintainable Saket's Language code. Remember, good code is like a good joke - if you have to explain it, it's probably not that good!

Do's

  • ‚úÖ Use descriptive variable names
  • ‚úÖ Add comments to explain complex logic
  • ‚úÖ Keep functions small and focused
  • ‚úÖ Use consistent indentation
  • ‚úÖ Test your mathematical calculations
  • ‚úÖ Handle edge cases gracefully

Don'ts

  • ‚ùå Don't use single-letter variable names
  • ‚ùå Don't write overly complex expressions
  • ‚ùå Don't ignore error handling
  • ‚ùå Don't hardcode values everywhere
  • ‚ùå Don't forget to test edge cases
  • ‚ùå Don't write code without comments

Clean Code Example

// Good: Descriptive names and clear structure
bhai studentGrades = [85, 92, 78, 96, 88, 91, 83]
bhai passingGrade = 75

// Calculate class statistics
calculate karo average(studentGrades)
bhai classAverage = result

calculate karo max(studentGrades)
bhai highestGrade = result

calculate karo min(studentGrades)
bhai lowestGrade = result

// Display results with clear formatting
bol "=== Class Performance Report ==="
bol "Total Students: " + studentGrades.length
bol "Class Average: " + classAverage.toFixed(2)
bol "Highest Grade: " + highestGrade
bol "Lowest Grade: " + lowestGrade

// Check passing rate
bhai passingStudents = 0
studentGrades.forEach(grade => {
    agar (grade >= passingGrade) {
        passingStudents = passingStudents + 1
    }
})

calculate karo (passingStudents / studentGrades.length) * 100
bhai passingRate = result

bol "Passing Rate: " + passingRate.toFixed(1) + "%"
